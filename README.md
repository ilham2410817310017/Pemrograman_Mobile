### Deskripsi Aplikasi
Aplikasi prototipe Android berbasis **Jetpack Compose** yang berfungsi untuk menampilkan informasi edukatif dan data teknis perangkat secara real-time. Aplikasi ini memiliki sistem navigasi **Horizontal Pager** dua halaman: 
1. **Halaman Sejarah Android**: Menyajikan garis waktu perkembangan Android dalam bentuk daftar yang dapat digulir.
2. **Halaman Info & Arsitektur**: Menampilkan visualisasi lima lapisan arsitektur sistem Android serta informasi perangkat dinamis seperti model ponsel, versi OS, status koneksi jaringan (Wi-Fi/Seluler), dan persentase baterai melalui integrasi API sistem.

---

### Cara Menjalankan
1. Pastikan perangkat Android sudah terhubung ke komputer menggunakan kabel USB.
2. Aktifkan **Opsi Pengembang (Developer Options)** dan fitur **Instal aplikasi melalui USB (Install via USB)** pada perangkat Android Anda.
3. Buka proyek di **Android Studio**.
4. Tunggu proses **Gradle Sync** hingga selesai.
5. Tekan tombol **Run/Play** (ikon segitiga hijau) untuk memasang dan menjalankan aplikasi pada perangkat.
